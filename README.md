# Movie Task

A React movie browser app with filtering by name, year, genre, and star rating. Users can like/unlike movies.

## Features
- Search movies by name
- Filter by year, genre, and star rating
- Like/unlike functionality
- Lazy-loaded poster images

## Installation
```bash
npm install
npm run dev
```
